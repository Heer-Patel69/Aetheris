# Aetheris User Documentation

Aetheris is an autonomous AI Engineering Operating System designed to plan, implement, test, and evolve software applications from high-level natural language requirements.

## How It Works
1. **Goal Input:** Provide a target product description (e.g. "Build a tasks scheduler").
2. **Autonomous Planning:** Aetheris creates requirements, designs architectures, selecting tech stacks automatically.
3. **Sandboxed Code Writing:** Writes modular Python and HTML source code inside a safe executor sandbox.
4. **Verification & Audit:** Runs unit tests and security reviews before completing.
