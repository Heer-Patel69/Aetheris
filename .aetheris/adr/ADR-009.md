# ADR-009: Enterprise Governance Artifacts Are Generated From Repository Evidence

Status: Accepted
Owner: Engineering Governance Board
Approval Date: 2026-07-01

## Context
Governance must stay synchronized with the repository.

## Problem
Manual governance pages drift quickly.

## Decision
Generate traceability, governance, ADR, drift, readiness, graph, metric, checklist, and report artifacts from source metadata.

## Alternatives
- Manual docs only
- External governance database

## Trade-Offs
- Generated docs can be repetitive
- Repeatable outputs are easy to validate

## Consequences
- Governance generation becomes part of release prep

- `scripts/generate_enterprise_governance.py`
