# ADR-008: Knowledge Retrieval And Verification Gate Engineering Claims

Status: Accepted
Owner: Principal AI Systems Engineer
Approval Date: 2026-07-01

## Context
Generated engineering guidance must remain grounded in repository evidence.

## Problem
Unsupported claims create hallucinated architecture and unsafe implementation steps.

## Decision
Chunk, retrieve, rank, verify, and scan claims before release to execution.

## Alternatives
- Trust model output
- Manual review only

## Trade-Offs
- Adds verification latency
- Improves auditability

## Consequences
- Claims require confidence and evidence references
