# ADR-007: Model And Prompt Intelligence Are Advisory, Measured, And Evidence-Gated

Status: Accepted
Owner: Principal AI Systems Engineer
Approval Date: 2026-07-01

## Context
Different engineering tasks require different context, quality, and cost profiles.

## Problem
Hard-coded model choice or loose prompts cause cost and quality drift.

## Decision
Use model registries, prompt compilation, optimization, structured reasoning, and self-reflection.

## Alternatives
- Single default model
- Prompt-only routing

## Trade-Offs
- Advisory routing may not force external tools
- Measured routing enables improvement

## Consequences
- Model choices must include confidence and cost evidence
