# ADR-011: Observability Is A First-Class Contract

Status: Accepted
Owner: Principal Infrastructure Engineer
Approval Date: 2026-07-01

## Context
Autonomous engineering needs measurable outcomes and debuggable failures.

## Problem
Opaque automation cannot be operated in production.

## Decision
Every subsystem must emit structured status, duration, warning, artifact, and failure telemetry.

## Alternatives
- Log free-form text only
- Rely on test output

## Trade-Offs
- More telemetry volume
- Better operations and readiness scoring

## Consequences
- Dashboards and readiness reviews consume generated metrics
