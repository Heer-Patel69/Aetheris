# Skill Usage Report

Aetheris loads and schedules 7 core skills:
1. `aetheris-kernel`
2. `aetheris-skill-orchestrator`
3. `aetheris-context-engine`
4. `aetheris-memory-engine`
5. `aetheris-product-intelligence`
6. `aetheris-project-discovery`
7. `aetheris-verification-engine`

All skills are registered and actively called within the planning and execution scheduler scopes.
