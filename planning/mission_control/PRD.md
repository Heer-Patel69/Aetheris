# Product Requirements Document (PRD)

## Vision
Aetheris Mission Control is a live engineering observability platform acting as the central nervous system for the Aetheris Engineering Hypervisor.

## Core Requirements
- Global, Workspace, and Session Dashboards
- Real-time telemetry via websocket streaming
- Execution replaying (Intent, Skills, RFCs, Models, Timeline, etc.)
- Support for 1500+ skills
- Zero simulated data; strict reliance on runtime state.
