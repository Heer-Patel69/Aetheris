# Aetheris Release Notes — Version 3.1.0

We are proud to release Aetheris OS v3.1.0, fully compliant with the ARB vNext baseline specification standard.

## New Capabilities
* **Core Sandbox Isolation:** Executes shell subprocesses inside isolated path perimeters.
* **Multi-Tenant RBAC:** Authorizes client requests using structured role levels (admin, viewer).
* **Autonomous Learning Loops:** Log experience data and mine patterns across runs.
* **Collaborative AI Agent Roles:** CEO, CTO, Architect, and Developer agents communicate via stateful steps.
* **Self-Evolution Cycles:** Orchestrates AST reviews, optimizations, and patch operations.
